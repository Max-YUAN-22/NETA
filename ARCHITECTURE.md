┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   GitHub Pages  │    │  ShinyApps.io   │    │   本地开发环境   │
│   (静态网站)     │    │  (Shiny应用)    │    │   (RStudio)     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
         └───────────────────────┼───────────────────────┘
                                 │
                    ┌─────────────────┐
                    │   NETA核心平台   │
                    └─────────────────┘
                                 │
         ┌───────────────────────┼───────────────────────┐
         │                       │                       │
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   数据层         │    │   分析层         │    │   展示层         │
│                 │    │                 │    │                 │
│ • GEO数据库      │    │ • 质量控制       │    │ • 交互式可视化   │
│ • TCGA数据库     │    │ • 数据标准化     │    │ • 图表生成       │
│ • 临床数据       │    │ • 批次校正       │    │ • 结果导出       │
│ • 基因注释       │    │ • 差异表达分析   │    │ • 用户界面       │

│                 │    │ • 生物标志物发现 │    │                 │
└─────────────────┘    └─────────────────┘    └─────────────────┘

数据流程:
原始数据 → 质量控制 → 标准化 → 批次校正 → 分析 → 可视化 → 结果

技术栈:
• 前端: Jekyll + GitHub Pages + CSS3
• 后端: R/Shiny + ShinyApps.io
• 数据处理: R + Bioconductor
• 数据库: GEO + TCGA + 本地存储
• 部署: Git + GitHub + rsconnect

主要功能模块:
1. 数据概览 - 样本统计和分布
2. 基因表达分析 - 热图和箱线图
3. 差异表达分析 - 火山图和结果表
4. 生存分析 - Kaplan-Meier曲线和Cox回归
5. 通路分析 - 富集分析和通路图
6. 数据下载 - 结果导出功能

用户工作流程:
1. 访问网站 → 2. 选择分析模块 → 3. 输入参数 → 4. 查看结果 → 5. 下载数据
